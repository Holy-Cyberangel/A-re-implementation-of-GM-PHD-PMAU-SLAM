# GM-PHD-PMAU-SLAM
**高斯混合概率假设密度 - 位姿地图交替更新 SLAM 算法复现**

基于位姿与地图轮换更新框架的 GM-PHD-PMAU-SLAM 算法复现，相比传统 RBPF 框架 PHD-SLAM 具有更低时间复杂度，有效抑制位姿与地图误差累积。

## 算法介绍
- 采用**位姿-地图轮换更新闭环框架**，仅维护单一位姿后验与一组地标，时间复杂度显著优于 RBPF 框架。
- 位姿更新阶段联合历史地图，减缓位姿误差累积。
- 以高精度位姿后验更新地标，降低地标估计误差，形成正向反馈闭环。
- 基于 GM-PHD 实现数据关联隐式处理，支持杂波、漏检等复杂观测场景。

## 复现流程

### 观测数据生成
- 仿真数据文件：`./GM-PHD-SLAM/data/Environment_SLAM.mat`
- 依赖真值：`Landmark_Groundtruth`、`Robot_Groundtruth`
- 量测噪声协方差（局部极坐标）：
  ```matlab
  R = diag([(0.8)^2, (0.3*pi/180)^2]);
  ```
- 模拟杂波（泊松分布数量 + 视域均匀分布）与漏检
- 输出：`measurement.mat`（含多组 Monte Carlo 量测与带噪声里程计）

### 初始化
- 执行参数初始化脚本
- 生成指定次数 Monte Carlo 观测数据
- 利用 k=1 时刻真值与观测初始化位姿后验与地图后验

### 算法框架
```
初始化 → 新生/剪枝/合并
→ 位姿预测(pose predict)
→ 位姿更新(pose update)
→ 地图预测(map predict)
→ 地图更新(map update)
→ 输出位姿后验/地图后验/轨迹估计/地图估计
```

#### 位姿预测
```matlab
[x_,P_] = pose_predict(x,P,u,dt,Qu)
```
- 模型：圆弧模型（连续运动积分解）
- 输入：上一时刻位姿、协方差、里程计、时间间隔、过程噪声协方差
- 输出：当前时刻位姿先验与协方差

#### 位姿更新
```matlab
[x,P] = pose_update(x_,P_,map,Z,params,model,R)
```
- 基于 PHD 隐式数据关联 + IEKF 迭代更新
- 融合策略：矩匹配（避免矩阵反复求逆）

#### 地图预测
```matlab
idx = map_predict(x_,map,params)
```
- 功能：划分视域内 / 视域外地标

#### 地图更新
```matlab
[map,birth_idx] = map_update(x,P,map,inmap,Z,params,R)
```
- 依据位姿后验更新视域内地标
- 新生策略：分量强度低于阈值则标记为新生地标，后续帧区分杂波与真实新地标

## 代码说明

### MEX 加速
- 无 MEX 版本：`./GM-PHD-PMAU-SLAM/`
- MEX 加速版本：`./GM-PHD-PMAU-SLAM-MEX/`
- 加速效果：提升 50~100 倍，单次 Monte Carlo 从 120s 降至 2s
- 注意：直接修改 MEX 文件夹内函数不生效，需重新编译

### 程序入口
- 主运行：`main.m`
  - 修改参数后直接运行，数据生成与调参已集成
- 可视化：`check.m`
  - 查看最后一次 Monte Carlo 轨迹、地图与真值对比
  - 并行模式需将 `parfor` 改为 `for`
- 数值分析：`data_analysis.m`
  - 计算 100 次实验指标均值与标准差
- 参数设置：`set_params.m`
  - 修改后直接生效，无需手动运行

---

## 使用方法
1. 配置 MATLAB 运行环境
2. 准备仿真数据：`Environment_SLAM.mat`
3. 在 `set_params.m` 中配置参数
4. 运行 `main.m` 开始复现
5. 运行 `check.m` 可视化结果
6. 运行 `data_analysis.m` 输出定量评估指标
